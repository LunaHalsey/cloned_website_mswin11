Det här är Windows 11

Intuitiv navigering. Enkel organisering. Snabbare prestanda. Dessutom nya funktioner för att låsa upp din produktivitet och kreativitet. Utforska dessa Windows 11-funktioner och ta reda på varför det är det bästa Windows hittills.

Lättare att använda

Utforska nya Windows 11-funktioner som hjälper dig att få saker gjorda snabbt, roligt och enkelt.

  • Allmän sökning

    Välj Sök i aktivitetsfältet och börja skriva det du letar efter. Du får anpassade resultat från dina filer, foton, kontakter, e-postmeddelanden och även webben.

  • Tillgänglighetsfunktioner

    Navigera med röstigenkänning1, se vad som sägs med livetextning eller få text på skärmen uppläst av en skärmläsare med en naturlig mänsklig röst.2 Windows 11 gör det enklare för alla att använda sin dator.

  • Enkel installation och anpassning

    Anpassa skrivbordstema, bakgrunder, färger, text och mycket mer för att skapa det perfekta kreativa utrymmet för att utforska din passion.

Lättare att vara produktiv

Skapa balans på ditt skrivbord med lättanvända verktyg och Windows 11-funktioner som hjälper dig att optimera ditt skärmutrymme och maximera din produktivitet.

  • Fästtips

    Organisera det som finns på din skärm i ett nafs för att ge plats åt dina bästa idéer. Dra bara ett fönster till skärmens kant för att aktivera fästtips och fäst dem på ett rent, organiserat rutnät.3

  • Skrivbordsgrupper

    Håll ditt digitala liv organiserat med skrivbordsgrupper för alla de olika sätt som du arbetar och spelar på. Anpassade bakgrunder och etiketter hjälper dig att skapa en öronmärkt skrivbordsyta för varje del av ditt liv.

  • Synkronisera din dator och telefon

    Microsoft Phone Link är nu tillgänglig för iPhone®-enheter och Android™. Visa meddelanden, läs meddelanden och ring samtal från din PC13. Mindre fram och tillbaka mellan datorn och telefonen innebär mer tid att fokusera.

Lättare att hålla sig säker

Funktionerna i Windows 11 ger dig kraften att skapa, samarbeta och hålla dina saker säkra.

  • Prestanda och säkerhet

    Från det ögonblick som du startar är Windows 11 på sin vakt. Det arbetar tillsammans med din maskinvara och har utformats med flera olika skyddslager för att säkra dina appar, din information och din integritet.5

  • Enhetsskydd med SmartApp Control

    Windows 11 kör endast appar med gott rykte och som inte kompromissar med dina data.6

  • Microsoft Edge-säkerhet

    Förutom att ge dig kontroll över dina data och skydda dig mot skadlig kod kan Microsoft Edge även skanna webben för att kontrollera om dina lösenord har komprometterats.

Lättare att spela

Spela de senaste spelen med grafik som utmanar verkligheten. Med funktionerna i Windows 11 kan du spela på ditt sätt.

  • PC Game Pass

    Spela hundratals högkvalitativa datorspel med vänner inklusive spel direkt på lanseringsdagen.7

  • Auto HDR

    Auto HDR gör det möjligt för dina spel att producera intensivare highlighter och skuggor på kompatibla skärmar.8

  • DirectStorage

    DirectStorage tillåter större världar med högre detaljrikedom att laddas snabbare och stöds nu av fler lagringskonfigurationer.9

Se minimisystemkraven för Windows 11
  • Simulerade skärmar. Tillgänglighet för funktioner och appar varierar beroende på region.
  • 1 Maskinvaruberoende. Kräver Windows-dator med mikrofonfunktion.
  • 2 Endast amerikansk engelska för livetextning och röståtkomst. Utbudet av naturliga röster varierar beroende på marknad: Se bilaga A: Språk och röster som stöds.
  • 3 Vissa layouter är endast tillgängliga baserat på skärmupplösning och skalningsinställningar.
  • 5 Under enhetssupportsperioden som erbjuds av tillverkaren av originalutrustningen (OEM, Original Equipment Manufacturer).
  • 6 Endast tillgänglig i den senaste versionen av Windows 11. För att hålla dig så säker som möjligt måste du börja med ett nytt system. Datorer som har kört tidigare versioner av Windows måste göra en fullständig återställning av operativsystemet Windows 11 för att funktionen ska fungera.
  • 7 Medlemskap säljs separat. Villkor och undantag gäller. Spelutbudet varierar över tid och beroende på region och enhet. Kräver Windows 10/11 (med uppdateringar); exkluderar S-läge och ARM-enheter. Mer information finns på https://www.xbox.com/pcgamepass, https://www.ea.com/ea-play/terms och https://www.ea.com/ea-play.
  • 8 Auto HDR kräver en HDR-skärm och stöds i titlar som använder DirectX 11 eller senare API.
  • 9 DirectStorage kräver API-anpassning av spel. Kräver en NVMe SSD för att kunna lagra och köra spel som använder drivrutinen Standard NVM Express Controller och en DirectX12 GPU med stöd för Shader Model 6.0.
  • 10 För datorer med Windows 10 i S-läge ska du ladda ned den här versionen av appen Datorhälsokontroll.
  • 11 Windows 11-uppgraderingen är tillgänglig för berättigade datorer som uppfyller minimispecifikationerna. Tidpunkten för uppgraderingen kan variera beroende på enhet. Internetavgifter kan tillkomma. Tillgänglighet för funktioner och appar varierar beroende på region. Vissa funktioner kräver specifik maskinvara (se specifikationerna för Windows 11).
  • 13 Phone Link-upplevelsen är förinstallerad på din dator med Windows 10 (med minst Windows 10, uppdatering maj 2019) eller Windows 11. För att få full funktionalitet måste Android-telefoner ha Android 7.0 eller senare. Phone Link för iOS kräver iPhone med iOS 14 eller högre, Windows 11-enhet, Bluetooth-anslutning och den senaste versionen av Phone Link-appen. Inte tillgängligt för iPad (iPadOS) eller MacOS®. Enheternas kompatibilitet kan variera. Regionala begränsningar kan förekomma. Varumärken tillhör sina respektive ägare.
